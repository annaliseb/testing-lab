## Quality Gates

All generated code must pass four gates, as described in the verify skill:

1. **Test Integrity** — test pass rate and coverage
2. **Health Check Completeness** — all health checks pass
3. **Deployment Readiness** — installable and runnable
4. **Maintainability** — code style and readability
